Funciones anónimas y Funciones flecha

Ejercicio

Crea una función que tome un número como argumento y:
Llama a la primera función 3 veces para probar todos los casos. Muestra los 3 resultados en consola.

  • Utilice una función anónima para verificar si el número es positivo, negativo o cero. Mostrará texto explicativo en consola.
  • Utilice una función flecha para elevar el número al cuadrado. Devolverá este resultado

Respuesta

                function eje5(num){
                    // primera funcion para verificar el numero
                    (function(n){
                        if(n > 0){
                            console.log(n + " numero es positivo");
                        }else if ( n < 0){
                            console.log(n + " numero es negativo");
                        }else{
                            console.log(n + " es cero");
                        }
                    })(num);

                    // funcion flecha
                    const cuadrado = n => n * n;
                    //Devolvera esta resultado
                    return cuadrado(num);
                }
                //llama la funcion
                const r1 = eje5(5);
                const r2 = eje5(-3);
                const r3 = eje5(0);
                // Muestrar 3 resultado en consola
                console.log("Cuadrado de 5:", r1);
                console.log("Cuadrado de -3:", r2);
                console.log("Cuadrado de 0:", r3);

Output

                5 numero es positivo
                -3 numero es negativo
                0 es cero
                Cuadrado de 5: 25
                Cuadrado de -3: 9
                Cuadrado de 0: 0
